inlineUploads

src/src/util/couchdbAttachments.js

Upload several attachments in one revision

inlineUploads(items: Array<object>, options: object?= ): Promise<object>
Parameters
items (Array<object>)
Name Description
items[].name string The name of the attachment
items[].contentType string The contentType of the uploaded data
items[].data string The attachment data to upload. If string, must be a valid base64 encoded dataURL.
items[].content string The attachment data to upload. Alias of data.
items[].file (Blob | string) The attachment data to upload. Alias of data.
options (object?= )
Returns
Promise<object>: The new list of attachments
Example
// With dataurl
cdb.inlineUploads([{
  name: 'example.png',
  file: 'data:image/png;base64,ORK5CYII='
}]);
// With Blob
cdb.inlineUploads([{
  name: 'example.txt',
  file: new Blob(['example'], {content_type: 'text/plain'});
}]);
// With data
cdb.inlineUploads([{
  name: 'example.txt',
  contentType: 'text/plain',
  data: 'example'
}]);
